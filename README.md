Je préviens d'office ça va pas être un read me diablissime,

L'idée du projet c'est pour une marque de connaître ses vrais ambassadeurs sur Instagram, qui sont les personnes qui commentent le plus régulièrement les posts de la marque et partagent les valeurs de la marque.
Ce que fait le projet : via la librairie Scrapfly on renseigne le compte Instagram à scrapper, le nombre de post qu'on veut scrapper (les n derniers posts) et le nombre de comptes identifiés commes les meilleurs ambassadeurs que l'on souhaite et le programme va livrer 3 fichiers excel.
Le top N des meilleurs ambassadeurs de la marque avec leur @, le nombre de commentaires, le nombre de posts commentés et le nombre de likes générés sur leurs commentaires.
Tous les comptes commenteurs d'au moins 1 post avec les mêmes KPI.
On peut aussi avoir la liste de tous les commentaires sur les n derniers posts pour ensuite analyser si ils correspondent aux valeurs de la marque.
